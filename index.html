🐼 Projeto Zoo Functions 🐾

📄 Sobre

Projeto do Bloco 8 - Higher Order Functions do JavaScript ES6 do curso de Desenvolvimento Web da Trybe.

Esse projeto simula um sistema de relatório de um zoológico que possui informações a respeito dos animais presentes no zoológico, colaboradores, horários de funcionamento e uma tabela de preços que varia de acordo com a idade das pessoas que o visitam.

Quantidade de requisitos: 10

Requisitos feitos: 10

💻 Habilidades desenvolvidas

  • Utilizar novas funcionalides do ES6, como arrow functions, template literals, spread operator, parâmetro rest e object destructuring;
  • Compor objetos e ler suas informações com diferentes métodos;
  • Manipular arrays com funções como map, filter e reduce.
